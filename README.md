# atomacao_arduino_esp32
# Automação Residencial com Arduino e ESP32

Bem-vindo ao repositório do nosso projeto de automação residencial! Este espaço foi criado para documentar nosso aprendizado prático com microcontroladores, sensores e Internet das Coisas (IoT).

## Integrantes
* **LUCAS SALES** - BoyHeavenSales
* **FRANCISCO PEDRO** - XICOVB

## Objetivo do Projeto
O objetivo principal é criar um sistema funcional de automação capaz de monitorar e controlar dispositivos domésticos, enquanto exploramos a integração entre o **Arduino** (controle local) e o **ESP32** (conectividade Wi-Fi).

## Stack Tecnológica
- **Hardware:**
  - ESP32 
  - Arduino 
  - Rele
- **Software:**
  - Linguagem C/C++
  - Arduino IDE
  - Wokwi

## Estrutura do Repositório
Para manter nossa organização, dividimos o projeto assim:
* `/arduino`: Códigos voltados para a lógica dos sensores e atuadores.
* `/esp32`: Códigos focados na conectividade e comunicação web.
* `/docs`: Esquemas elétricos, fotos do protótipo e links úteis.
